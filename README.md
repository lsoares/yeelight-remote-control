# Yeelight Remote Control

## Features
- Power on/off
- Set brightness
- Switch between Normal/Moonlight mode
- Set temperature

## Building
1. git clone this project
2. Run:
```
dotnet build
```
3. go to `\bin\Release\netcoreapp3.1`
